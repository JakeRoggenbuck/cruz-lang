# cruz-lang

Cruz Lang is a simple explicit static typed language. <br>
The compiler is called surf. <br>
It has 5 primitive data types:
```
Byte (1 char)
Char (1 byte)
Int (2 bytes)
Prec (4 bytes) 
Ptr (4 byte)
```
